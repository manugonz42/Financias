# Assets

This directory contains images and screenshots for the Financias project.

## Contents

- `dashboard.png` — Main dashboard screenshot
- `import.png` — Import screen screenshot
- `charts.png` — Charts and graphs screenshot

## Specifications

| Image | Size | Format | Description |
|-------|------|--------|-------------|
| dashboard.png | 1280×720 | PNG | Main dashboard view |
| import.png | 1280×720 | PNG | PDF import screen |
| charts.png | 1280×720 | PNG | Financial charts |

## How to Capture

1. Run the app: `npm run tauri dev`
2. Navigate to the screen you want to capture
3. Take a screenshot (Windows: `Win + Shift + S`)
4. Save the image in this directory
5. Update the README.md to reference the new image

## Optimization

- Use PNG format for screenshots
- Keep images under 1MB
- Compress with [TinyPNG](https://tinypng.com) if needed
